in quanto la versione che viene installata è datata (3.6.x).
Per far interagire Node.js con MongoDB occorre una versione di
MongoDB >= v 4.0.0.

MondoDB 7.x.x all'interno della macchina virtuale.

Ti spiego come installare MongoDB su una macchina virtuale Ubuntu passo-passo.

Assumiamo che tu stia usando Ubuntu 20.04 o 22.04 su Vagrant (o comunque una macchina Ubuntu a 64 bit). I comandi vanno eseguiti dentro la VM, quindi prima accedi con:

bash
Copy
Edit
vagrant ssh
🔧 1. Importa la chiave pubblica GPG di MongoDB
bash
Copy
Edit
curl -fsSL https://pgp.mongodb.com/server-7.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-7.0.gpg --dearmor
📦 2. Aggiungi il repository ufficiale
Per Ubuntu 22.04 (Jammy):

bash
Copy
Edit
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list
Per Ubuntu 20.04 (Focal):

bash
Copy
Edit
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list
🔄 3. Aggiorna i pacchetti
bash
Copy
Edit
sudo apt update
📥 4. Installa MongoDB
bash
Copy
Edit
sudo apt install -y mongodb-org
▶️ 5. Avvia e abilita MongoDB come servizio
bash
Copy
Edit
sudo systemctl start mongod
sudo systemctl enable mongod
Puoi controllare lo stato con:

bash
Copy
Edit
